Getting started with CSS

Introduction to CSS

•  What is CSS? CSS (Cascading Style Sheets) is a stylesheet language used to describe the presentation of a document written in HTML or XML. It defines how elements on a webpage should be displayed, including aspects like layout, colors, fonts, spacing, and positioning.

Why is CSS Important?

  • Improves User Experience: CSS makes websites look better and easier to use by controlling things like colors, fonts, and layout, helping users navigate and enjoy the site more.
  • Separation of Content and Style: With CSS, you can change the design of a website without changing the content, making it easier to update and maintain.
  • Responsive Design:CSS allows websites to adjust their layout for different screen sizes, ensuring they look good on all devices, from phones to desktops.

    • Why you should learn Css

Main concepts:

  1. Selectors: These are used to target HTML elements you want to style. You can use element selectors (like h1), class selectors (like .class-name), or ID selectors (like #id-name).

    2. •   Examples:
    1. Element Selector: Targets all <h1> elements.

      2. css 
      
    h1 {                                                             
      
        color: blue;  /* Applies to all h1 elements */               
      
      }

    2. Class Selector: Targets all elements with a specific class (.example).

      3. css 
      
    .example {                           
      
        background-color: lightgray;  /* Applies to all elements with class "example" */
      
      }

    3. ID Selector: Targets a specific element with a unique ID (#unique).

      4. css 
      
    #unique {           
      
        font-size: 20px;  /* Applies to the element with ID "unique" */   
      
      }


  2. Properties and Values: These define the style you want to apply. For example, the color property sets text color, and font-size changes the size of the text. Each property has a value (e.g.,color: red;).

    3. •   Examples:
    1. Changing Text Color and Size:

      2. css 
      
    p {
      
        color: green;  /* Sets text color to green */
      
        font-size: 18px;  /* Sets font size to 18px */
      
      }

    2. Background Color:

      3. css 
      
    div {
      
        background-color: lightblue;  /* Sets the background color to light blue */
      
      }


  3. Box Model: Every HTML element is a rectangular box, and the box model determines its size and spacing. It includes the content area, padding, border, and margin.

    4. •   Example (Content, Padding, Border, and Margin):
    css 
    
        .box {
    
            width: 200px;
    
            height: 150px;
    
            padding: 10px;  /* Adds space inside the box */
    
            border: 5px solid black;  /* Adds a border around the box */
    
            margin: 20px;  /* Adds space outside the box */
    
          }



  4. Positioning: CSS lets you control the position of elements on a page with properties like position, top, left, right, and bottom. This helps you control layout and placement of elements.

    5. •   Examples:
    1. Relative Positioning:

      2. css 
      
        .relative-box {
      
            position: relative;
      
            top: 20px;  /* Moves the element 20px down from its original position */
      
            left: 10px;  /* Moves the element 10px to the right */
      
          }

    2. Absolute Positioning:

      3. css 
      
        .absolute-box {
      
            position: absolute;
      
            top: 50px;  /* Position 50px from the top of the nearest positioned ancestor */
      
            left: 100px;  /* Position 100px from the left of the nearest positioned ancestor */
      
          }


  5. Flexbox: A layout model in CSS that makes it easy to align and distribute space among items in a container, even when their sizes are unknown or dynamic.

    6. •   Example (Basic Flexbox Layout):
    css 
    
            .container {
    
                display: flex;  /* Turns the container into a flex container */
    
                justify-content: space-between;  /* Spaces items evenly */
    
                align-items: center;  /* Vertically aligns items in the center */
    
            }
    
              
    
              .item {
    
                background-color: lightgreen;
    
                padding: 10px;
    
              }



  6. Media Queries: These are used to apply different styles depending on the screen size or device, helping create responsive web designs that work on both mobile and desktop screens.

    7. •   Example (Responsive Design: Change styles based on screen width):
    css 
    
            @media (max-width: 768px) {
    
                body {
    
                  font-size: 14px;  /* Decreases font size for screens smaller than 768px */
    
                }
    
              
    
                .container {
    
                  flex-direction: column;  /* Changes the layout from row to column on smaller screens */
    
                }
    
              }
