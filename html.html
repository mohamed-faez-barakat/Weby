Getting started with HTML

Introduction to HTML

•  What is HTML? HTML (HyperText Markup Language) is the foundation of web development. It is the standard language used to create and structure content on the web. HTML defines the structure of web pages using a system of tags that describe elements such as text, images, links, and multimedia. Understanding HTML is the first step toward building websites and applications.

Why is HTML Important?

  • Structure: HTML structures the content you see on a webpage (e.g., headings, paragraphs, images).
  • Interactivity: It forms the base for adding interactive features when combined with CSS and JavaScript.
  • Accessibility: A properly structured HTML document ensures that web content is accessible and readable by all users, including those with disabilities.

Why you should learn HTML

Main concepts:

  1. HTML Tags HTML tags are the fundamental building blocks of the language. Tags are enclosed in angle brackets (< >) and come in pairs, where the first tag is the opening tag and the second tag is the closing tag. The closing tag has a forward slash before the tag name (e.g., </p>).
    2. •   Example of a simple tag:
    html 
    <h1>Welcome to Weby!</h1>


  2. HTML Elements An HTML element is a combination of an opening tag, content, and a closing tag. Elements tell the browser what type of content they contain (such as text, an image, or a link) and how it should be displayed.
    3. •   Example:
    html 
    <p>This is a paragraph of text.</p>

    The <p> tag defines a paragraph, and everything between the opening <p> and closing </p> is considered part of that element.



  3. HTML Attributes Attributes provide additional information about an element. They are always written in the opening tag and provide further details such as the style, source, or behavior of an element. Common attributes include class, id, src, and href.
    4. •   Example of an image element with attributes:
    html 
    
<img src="image.jpg" alt="A description of the image" width="300" height="200">

    • src: Defines the image source (file path or URL).

    • alt: Provides alternative text that describes the image.

    • width and height: Control the size of the image.

Simple HTML Examples

1- Example 1: Basic Page StructureA simple HTML page is made up of the following structure:

  • The <!DOCTYPE html> declaration specifies the document type (HTML5).

  • The <html> tag is the root element of the page.

  • The <head> section includes metadata such as the title and character encoding.

  • The <body> section contains the actual content displayed on the page.

•Example:

html 

<!DOCTYPE html>

<html lang="en">

<head>

    <meta charset="UTF-8">

    <meta name="viewport" content="width=device-width, initial-scale=1.0">

    <title>My First HTML Page</title>

</head>

<body>

    <h1>Welcome to Weby</h1>

    <p>This is a simple HTML page demonstrating the structure of a website.</p>

</body>

</html>


2- Example 2: Adding Links Hyperlinks allow users to navigate between pages or to external websites. To create a link, we use the <a> tag and the href attribute, which specifies the destination URL.



• Example:

html 
<a href="https://www.google.com">open google</a>

•href: The link destination.
