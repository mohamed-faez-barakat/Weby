Getting started with JavaScript

Introduction to JavaScript

•  What is JavaScript? JavaScript is a programming language that helps make websites interactive. While HTML builds the structure of a webpage and CSS styles it, JavaScript adds actions and behaviors. For example, it can make things move on the screen, respond when you click a button, or show new content without reloading the page. JavaScript is important because it helps bring web pages to life, making them more fun and useful for users.

Why is JavaScript Important?

  • Interactivity: JavaScript allows websites to respond to user actions, like clicking buttons or submitting forms.
  • Dynamic Content: With JavaScript, web pages can update content without needing to reload the entire page, creating smoother experiences.
  • Widely Used: JavaScript is essential in web development, making it a key skill for building modern, interactive websites.
  • Versatility: JavaScript can be used for everything from simple website features to complex web applications, making it one of the most versatile programming languages.

Why you should learn JavaScript

Main concepts:

  1. Variables Variables store data, such as numbers, text, or other values, that you can use and change in your program. In JavaScript, you create variables using let, const, or var.
    2. •   Example:
    javascript 
    
    let name = "John";  // This is a string variable
    
let age = 25;       // This is a number variable


  2. Data Types JavaScript has different types of data, like strings (text), numbers, booleans (true/false), arrays (lists of values), and objects (collections of key-value pairs).
    3. •   Example:
    javascript 
    
    let isStudent = true;  // Boolean 
    
    let message = "Hello!"; // String 
    
    let score = 90;         // Number


  3. Functions Functions are blocks of code that perform a specific task. You can define a function once and use it multiple times to keep your code organized and reusable.
    4. •   Example:
    javascript 
    
    function greet() { 
    
        alert("Hello, World!");  // Displays a greeting  
    
    }  
    
    greet();  // Calling the function

  4. Events Events are actions that happen on a webpage, like a mouse click or a key press. JavaScript allows you to listen for these events and respond to them, like changing text when a button is clicked.
    5. •   Example:
    javascript 
    

    document.getElementById("myButton").onclick = function() { 
    
        alert("Button clicked!");   
    
    };

  5. Conditionals Conditional statements let you make decisions in your code. You can use if, else, and switch to control what happens based on different conditions.
    6. •   Example:
    javascript 
    

    let age = 20;
    
    if (age >= 18) {  
    
        alert("You are an adult.");  
    
    }  
    
     else {  
    
        alert("You are a minor.");  
    
    }
